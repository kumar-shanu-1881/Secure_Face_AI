About SecureFace AI

SecureFace AI is an AI-based face authentication system developed to provide secure user verification using facial recognition. The system detects user at every 350 ms. The system generates facial embeddings using InsightFace Buffalo_SC, verifies passwords using Scrypt hashing, and protects against spoofing attacks using an anti-spoof detection pipeline.
